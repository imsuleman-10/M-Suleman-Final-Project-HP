# 💻 Hunarmand - Frontend

This is the frontend component of the **Hunarmand LMS**, built with **React** and **Vite**. It provides a high-performance, responsive user interface with a glassmorphism design.

## 🚀 Key Technologies
- **React 19**: Modern UI development.
- **Vite**: Ultra-fast build tool and dev server.
- **Bootstrap & React-Bootstrap**: Responsive styling and layout.
- **Lucide React**: Premium icon library.
- **Axios**: API communication.
- **React Router 7**: Sophisticated client-side routing.

## 📁 Folder Structure
```text
src/
├── components/ # Reusable UI components (Navbar, CourseCard, etc.)
├── context/    # Global state management using Context API (AuthContext)
├── pages/      # Main page containers (Home, Dashboard, Courses)
├── services/   # API service functions for backend communication
├── App.jsx     # Main application component and routing configuration
└── main.jsx    # Application entry point
```

## ⚙️ Installation & Development

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:5173`.

3. **Build for Production**
   ```bash
   npm run build
   ```
